# Optimisation Métaheuristique du Placement des Données dans les Clusters Hadoop Distribués

## Membres
- Ali Nouar
- Souhail El Bettachi
- Mohammed Bekkouri

## Technologies
- Hadoop 3.3.6
- HDFS
- Python
- NumPy
- Pandas
- Matplotlib

## Objectif
Optimiser la répartition des blocs de données dans un cluster Hadoop à l'aide d'un algorithme génétique et comparer les résultats avec HDFS par défaut et HDFS Balancer ,AG et PSO..
